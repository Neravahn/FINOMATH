# FINOMATH 
FINOMATH is a web-based financial calculator designed to help users plan, compare, and analyze financial decisions such as interest calculations, annuities, present value, and future value.
It is built with HTML, CSS, and Vanilla JavaScript, focusing on clarity, correctness, and extensibility



## Features

- Simple Interest Calculator
- Compound Interest Calculator
- Present Value (Annuity) Calculator
  - Regular
  - Immediate

- Future Value (Annuity) Calculator
  - Regular
  - Immediate

- Multiple compounding intervals:
  - Yearly
  - Half-Yearly
  - Quarterly
  - Monthly

-  Clean, dark-themed UI
-  Dynamic form rendering based on selection
- Accurate time calculation using logarithmic formulas


## How It Works

1. User selects a calculation type from the dropdown.
2. Relevant input form is rendered dynamically.
3. User enters values and selects compounding interval.
4. Clicking GET RESULT calculates and displays the time period.
5. Changing any input clears the result to prevent stale values.
